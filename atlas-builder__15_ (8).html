Atlas Builder

Select your atlas folder. It should contain a plan file whose name ends in "Plan" (like Tree Atlas Plan.txt) and your media files — photos, audio, video, text, or HTML. Atlas Builder reads the plan, finds the media, and produces a single self-contained HTML atlas. Note: audio and video files greatly increase atlas size and load time. Use them sparingly, only where they make a real difference to the reader's experience.

1
Select your atlas folder

Choose the folder that contains your plan file(s) and media folders/files. When the file picker opens, navigate to your atlas folder and click it. If you have any media folders in it, they'll show; if not, the file picker will show nothing, and complain that "No items match your search". Ignore the message and tap the Upload or Select button at lower-right. Then confirm that you trust the site. I will show a summary of what has been uploaded.

2
Which file is your plan?

Choose the plan file from the list. It's the .txt file that defines your atlas columns, rows, and captions.

3
Choose build mode

Offline: all media embedded in one self-contained HTML file. Large file, slow to open, but works anywhere with no internet.
Online: small HTML file that downloads media from a server when opened. User needs wifi for initial load, then works offline until the tab is closed.

Enter the base URL where your media files will be hosted. The atlas will fetch files like baseURL/Cardon.tree.jpg.

4
Build your atlas

Atlas Builder will match media to columns and rows and build your atlas. A build report will show any issues found.
